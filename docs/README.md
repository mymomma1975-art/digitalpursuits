# NexusCommand Documentation

This directory contains all documentation, guides, and resources for the NexusCommand platform.

## Directories

### `/guides/`
- **NexusCommand-Complete-User-Guide.md** — Comprehensive guide covering all features, building agents, creating websites, implementing analytics
- **NexusCommand-Client-Intake-Form.md** — Professional client intake form for gathering requirements
- **NexusCommand-Deployment-Guide.md** — Step-by-step deployment guide for AWS and custom domains
- **deployment-infographic.png** — Visual architecture showing deployment options
- **hosting-workflow.png** — 6-step workflow from intake to go-live
- **domain-connection.png** — Domain connection process diagram

### `/presentations/`
- **nexuscommand-client-presentation/** — Professional client-facing slide deck (14 slides)
  - Covers platform features, benefits, pricing, deployment options, and call to action
  - Designed for sales presentations to potential clients

## Quick Start

1. **For Sales:** Use the client presentation to showcase NexusCommand to potential clients
2. **For Setup:** Follow the Deployment Guide to deploy your first instance
3. **For Clients:** Share the User Guide and Intake Form with clients to gather requirements
4. **For Operations:** Reference the infographics for deployment architecture and workflow
